# TODO アプリ

シンプルで使いやすいWebベースのTODOリスト管理アプリケーションです。

## 機能

### 基本機能
- ✅ タスクの追加・削除
- ✅ タスクの完了/未完了の切り替え
- ✅ タスクの編集（ダブルクリックまたは編集ボタン）
- ✅ LocalStorageで自動保存（ブラウザを閉じてもデータが残ります）

### 便利機能
- 🔍 フィルタリング
  - すべて：全タスクを表示
  - 未完了：未完了のタスクのみ表示
  - 完了済み：完了したタスクのみ表示
- 📊 タスク数の統計表示
- 🗑️ 完了済みタスクの一括削除
- 📱 レスポンシブデザイン（モバイル対応）

## 使い方

### 起動方法
1. `index.html` をブラウザで開く
   - ファイルをダブルクリック、または
   - ブラウザにドラッグ&ドロップ

### 操作方法

#### タスクの追加
1. 入力欄に新しいタスクを入力
2. 「追加」ボタンをクリック、またはEnterキーを押す

#### タスクの完了
- チェックボックスをクリックして完了/未完了を切り替え

#### タスクの編集
- タスクのテキストをダブルクリック、または編集ボタン（✏️）をクリック
- 編集後、Enterキーで確定、Escキーでキャンセル

#### タスクの削除
- 削除ボタン（🗑️）をクリック

#### フィルタリング
- 画面上部のボタンで表示するタスクを切り替え
  - 「すべて」：全タスク
  - 「未完了」：未完了のタスクのみ
  - 「完了済み」：完了したタスクのみ

#### 完了済みタスクの削除
- 「完了済みを削除」をクリックして、完了したタスクを一括削除

## 技術仕様

### 使用技術
- HTML5
- CSS3（レスポンシブデザイン）
- Vanilla JavaScript（フレームワーク不要）
- LocalStorage API

### ブラウザ対応
- Chrome/Edge（最新版）
- Firefox（最新版）
- Safari（最新版）

### データ保存
- LocalStorageを使用して自動保存
- ブラウザのデータを削除しない限り、データは永続的に保存されます

## カスタマイズ

### 色の変更
`style.css` の以下の部分を編集：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 最大表示件数の変更
`style.css` の以下の部分を編集：
```css
.todo-list {
    max-height: 400px;
}
```
